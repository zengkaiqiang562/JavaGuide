## 1. `linux` 目录结构

### 1.1 命令目录：`/bin --> /usr/bin`、`/sbin --> /usr/sbin`

### 1.2 用户家目录：`/home`、`/root`

### 1.3 配置文件目录：`/etc`

### 1.4 启动目录：`/boot`

### 1.5 设备目录：`/dev`

#### 1.5.1 黑洞：`/dev/null`

#### 1.5.2 产生随机数：`dev/random`

### 1.6 临时目录：`/tmp`

### 1.7 可变目录：`/var`

### 1.8 运行时目录：`/proc`

### 1.9 系统目录：`/usr`

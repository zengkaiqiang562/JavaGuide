## 1. 字节码执行引擎概述

`JVM` 字节码执行引擎的功能基本就是：
1. 输入字节码文件；
2. 然后对字节码进行解析并处理；
3. 最后输出执行的结果。

`JVM` 字节码执行引擎的实现方式可以是：
1. 通过解释器直接解释执行字节码；
2. 或者通过即时编译器产生本地代码，也就是编译执行。
3. 也可能是以上两者皆有。
